# Vulktek Cybersecurity - Web Portal

This repository contains the source code for the Vulktek Cybersecurity landing page. 

It is built as a highly optimized React application and is configured to deploy directly to Cloudflare's Edge Network.

## Deployment
This project is configured for seamless deployment to Cloudflare. Pushing to the main branch will automatically trigger a build and deployment if connected to Cloudflare Pages/Workers.

## Local Development
To run the site locally on your machine for testing:
1. Open your terminal in this directory.
2. Run `npm install` to ensure all dependencies are downloaded.
3. Run `npm run dev` to start the local server.
4. Open your browser to the provided localhost URL.
